== [[RandomForestRegressor]] RandomForestRegressor

`RandomForestRegressor` is a link:spark-mllib-Predictor.md[Predictor] for http://en.wikipedia.org/wiki/Random_forest[Random Forest] machine learning algorithm that trains a `RandomForestRegressionModel`.

[source, scala]

import org.apache.spark.mllib.linalg.Vectors
val features = Vectors.sparse(10, Seq((2, 0.2), (4, 0.4)))

val data = (0.0 to 4.0 by 1).map(d => (d, features)).toDF("label", "features")
// data.as[LabeledPoint]
